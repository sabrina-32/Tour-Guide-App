# Tour-Guide-App
# Android app project SWE250 

Features included:
------------------
1) Authentication (using Firebase)
2) Creating Post 
3) Search by Location & Type (Food , Hotel, Places)
4) Show in map (radius 10 km)
5) Showed in Detailed Post 
6) Radius Fixation
7) Fragmented UI(Tab switching)
